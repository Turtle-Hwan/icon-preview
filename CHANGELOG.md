# Changelog

All notable changes to the "icon-preview" extension will be documented in this file.

## [0.1.0] - 2024-12-24

### Added
- Initial release
- Gutter and inline icon preview
- Lucide icon URL support (automatic CDN conversion)
- JSDoc `@preview` tag support
- Configurable icon size, color, and position
- SVG `currentColor` replacement
- Image caching for performance
